# Homebrew Tap for LeanCTX

Hybrid Context Optimizer — Shell Hook + MCP Server for AI coding tools.

## Installation

```bash
brew tap yvgude/lean-ctx
brew install lean-ctx
```

Or in a single command:

```bash
brew install yvgude/lean-ctx/lean-ctx
```

## Updating

```bash
brew update && brew upgrade lean-ctx
```

## Uninstalling

```bash
brew uninstall lean-ctx
brew untap yvgude/lean-ctx
```
